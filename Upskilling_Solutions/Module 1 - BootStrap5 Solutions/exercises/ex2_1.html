Bootstrap 5 Folder Structure Explained

Understanding the core folders in a downloaded Bootstrap package.

/css Folder
Stylesheet Assets

Contains the compiled CSS files. The primary file is bootstrap.css (or the minified version bootstrap.min.css).

  • bootstrap.css: Main stylesheet.
  • bootstrap.grid.css: Only the flexbox grid system.
  • bootstrap.utilities.css: Only utility helper classes.
/js Folder
Javascript Plugins

Contains the compiled JS files that power interactive components like modals, tooltips, dropdowns, and accordions.

  • bootstrap.js: Main plugins.
  • bootstrap.bundle.js: Includes Popper.js for tooltips/dropdowns.
  • bootstrap.esm.js: ECMAScript Module format.
/icons Folder
Bootstrap Icons

A separate library containing thousands of SVG icons. Can be installed via npm or linked via CDN to render icons.

  • bootstrap-icons.css: Standard web font link.
  • SVG sprites: Inline SVG code integration.
  • SVG files: Direct image tag mapping.
