Setting up Bootstrap 5 with Sass

A step-by-step developer guide to initializing an NPM workspace and compiling custom SASS configurations.

Step 1: Initialize Node Project

Create your project directory and run the NPM initialization utility to generate your package.json file. 

npm init -y
Step 2: Install Sass and Bootstrap

Download and install the official Sass compiler (Dart Sass) and the Bootstrap source assets. 

npm install sass bootstrap --save-dev
Step 3: Define Compile Scripts

Open package.json and append the compilation command inside the scripts block. 

"scripts": {
  "build-css": "sass src/scss/custom.scss dist/css/custom.css",
  "watch-css": "sass src/scss/custom.scss dist/css/custom.css --watch"
}
Step 4: Structure SCSS Files

Create src/scss/custom.scss. Map it to import custom variable overrides first, followed by the Bootstrap source: 

// 1. Custom variables (e.g. overrides)
$primary: #6f42c1;
$border-radius: 12px;

// 2. Import Bootstrap source
@import "node_modules/bootstrap/scss/bootstrap";
💡 Why use Sass?

Sass allows you to modify Bootstrap's built-in global variables directly before compilation. It avoids bloating your code with CSS overrides and generates clean, optimized stylesheets.
